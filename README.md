# Scribbit
Scribbit is a simple, lightweight, and minimal text editor for Linux.

# Installation
## Arch
### AUR
#### PKGBUILD
```
git clone https://aur.archlinux.org/scribbit.git
cd scribbit
makepkg -si
cd ..
rm -rf scribbit
```
#### Yay
```
yay scribbit
```
